# Generated output

This directory is generated by workbench and experiment commands. Only this policy file is committed.

Build and reproduce the Sprint 1 comparison from the repository root:

```sh
npm ci
npm run build
node_modules/.bin/music experiment run \
  experiments/definitions/spatial-melody-comparison.experiment.json \
  --out output/spatial-melody-comparison
```

The command writes a deterministic top-level comparison page, `experiment-run.json`, aggregate
diagnostics, and `explicit-grid/` and `proportional-spatial-melody/` treatment bundles. Generated
files remain ignored; delete the output directory or run into a clean directory to prove byte
identity independently.
