# Coding Standards

Status: Architecture Sprint 0 must finalize.

Minimum expectations:

- TypeScript strict mode;
- deterministic behavior;
- explicit domain types;
- no musical semantics embedded only in renderer code;
- no unescaped user text in HTML;
- plain-language diagnostics;
- tests for invalid states;
- stable IDs and provenance;
- public types documented;
- examples for non-obvious schema fields;
- every defect discovered in corpus use becomes a regression test;
- prototype code may be replaced, but findings and canonical fixtures must remain durable.
